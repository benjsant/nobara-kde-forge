Ecran de connexion (Plasma Login)

Applique le theme, curseur et numlock a l'ecran de connexion plasma-login-manager (DM par defaut de Nobara/Fedora KDE 42+) via la configuration dans /etc/plasmalogin.conf.d/.

Les themes doivent etre installes dans /usr/share/themes ou /usr/share/icons (pas dans ~/.themes) car le display manager tourne en tant qu'utilisateur systeme. Utilisez l'option "Installation systeme" dans le catalogue de themes.
